# Student Management System

A responsive Student Management System (SMS) built with HTML, CSS, and JavaScript. Manage student records efficiently by adding, editing, and deleting entries dynamically. Displays student name, age, marks, result, and total count with a clean, mobile-friendly interface.

## Features

- Add students with name, age, and marks.
- Automatically display result based on marks.
- Edit or delete student records directly from the table.
- Shows total number of students.
- Responsive design for mobile, tablet, and desktop.
- Real-time form validation to ensure proper input.
- Smooth animations for UI elements and table rows.

## Technologies Used

- HTML5 – Structure and forms
- CSS3 – Styling, animations, responsive design
- JavaScript – DOM manipulation and dynamic table updates
